Django项目简记

先决条件：

1.一台linux服务器（可以用虚拟机代替）

2.配置docker环境（前期可以省略）

1.创建项目

django-admin startproject "test"

2.建立git仓库

cd test
git init
git add .
git commit -m "zi ji xiang yi ge mingzi"
// 然后用远程仓库进行连接，上传保存到云端

3.进行在线测试

cd test 
python3 manage,py runserver 0.0.0.0:8000
cd test/test 
vim settings 
// 将自己服务器的ip地址放到allowed_host里面

4.添加gitignore文件

cd test 
vim .gitignore
输入以下内容：
**/__pycache__
*.swap

5.更新数据库

python3 manage.py migrate

6.创建超级用户

python3 manage.py createsuperuser
// 输入以下内容
username
email
password
repeat password

7.建立app

cd test
python3 manage.py stratapp application

8.建立必要的文件夹

cd DjangoLearning/application
mkdir templates urls views static
// 在上面所有的python文件里面都要创建一个__init__.py文件

9.设置时区

cd DjangoLearning/DjangoLearning
vim settings.py
TIME_ZONE = 'Asia/Shanghai'

10.注册App

cd DjangoLearning/DjangoLearning
vim settings.py
installed_apps = ['application.apps.ApplicationConfig']

11.设置静态文件路径

STATIC_ROOT = os.path.join(BASE_DIR, 'static')
STATIC_URL = '/static/'

MEDIA_ROOT = os.path.join(BASE_DIR, 'media')
MEDIA_URL = '/media/'

// 可以将开发者的文件存在static里面，将用户的文件存在meida文件里面

cd application

cd menu 
wget --output-document=自定义图片文件名  图片地址

13.编写打包的脚本文件

cd DjangoLearning
mkdir scripts
cd scripts
vim compress_game_js.sh
// 输入以下内容
#! /bin/bash

JS_PATH=/home/gujin/DjangoLearning/application/static/js/
JS_PATH_DIST=${JS_PATH}dist/
JS_PATH_SRC=${JS_PATH}src/

find $JS_PATH_SRC -type f -name '*.js' | sort | xargs cat > ${JS_PATH_DIST}game.js
// 保存并退出
chmod +x compress_game_js.sh
